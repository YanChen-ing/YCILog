获取统一的设备id，基于UICKeyChain

## 特性

- [x] 生成获取存储在 keychain 中的 设备id  
- [x] 钥匙串中 id 相对稳定， 在不切换 appleID 的情况下，不会改变。


## 安装

```ruby
pod 'YCIDeviceId'
```
